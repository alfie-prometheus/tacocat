The Code for Taco Cat


                        function cleanString(input) {
                            return input.toLowerCase().replace(/[^a-z]/g, ''); // convert to lowercase and remove non-alphabetic characters
                        }
                        
                        function getInput() {
                            let inputWord = document.getElementById("inputWord").value;
                            return cleanString(inputWord);
                        }
                        
                        function tacoCat(inputWord) {
                            if (!inputWord) {
                                console.log("You must provide an input.");
                                return;  // Make sure to exit the function if there's no input
                            }
                        
                            const letters = inputWord.split("");
                            const n = letters.length;
                            let isPalindrome = true;
                        
                            for (let i = 0; i <= n / 2; i++) {
                                if (letters[i] !== letters[n - i - 1]) {
                                    isPalindrome = false;
                                    break;
                                }
                            }
                        
                            if (isPalindrome) {
                                displayResult("Success! " + inputWord + " is a Palindrome!");
                            } else {
                                displayResult("" + inputWord + " is not a Palindrome.");
                            }
                        }
                        
                        function displayResult(message) {
                            document.getElementById("result").innerHTML = message;
                        }
                        
                        document.getElementById('inputSubmit').addEventListener('click', function () {
                            const input = getInput();
                            tacoCat(input);
                        });
Taco Cat

The code comprises several functions aiming to determine if a provided string,
taken from a webpage input field labeled "inputWord", is a palindrome:
cleanString(input): This function processes the given string. It converts the string
to lowercase and then removes any characters that aren't in the alphabet.
getInput(): Fetches the user's input from the webpage's input field identified by "inputWord". After retrieving, it cleans the input using the cleanString function.
tacoCat(inputWord): Determines if the cleaned input string is a palindrome. A palindrome is a word, phrase, number, or another sequence of characters that reads the same forward and backward (ignoring spaces, punctuation, and capitalization). If the string is found to be a palindrome, a success message is displayed. If not, a failure message is shown.
displayResult(message): Handles the display of results on the webpage. The provided message is injected into an HTML element with the ID "result".
Lastly, an event listener is attached to a button with the ID "inputSubmit". When this button is clicked,
the palindrome check is triggered using the provided input string.
